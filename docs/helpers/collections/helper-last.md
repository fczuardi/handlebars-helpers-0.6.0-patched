#### \{{last}}
_Returns the last item in a collection. Opposite of `first`._
<br>Parameters: `none`

Data:

```json
"collection": [
  "Amy Wong",
  "Bender",
  "Dr. Zoidberg",
  "Fry",
  "Hermes Conrad",
  "Leela",
  "Professor Farnsworth",
  "Scruffy"
]
```
Template:

```html
\{{last collection}}
```

Renders to:

```
Scruffy
```
