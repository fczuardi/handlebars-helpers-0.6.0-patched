#### \{{ordinalize}}
_Turns a number into an ordinal string. Taken from the templating library [Walrus](https://github.com/jeremyruppel/walrus) by [Jeremy Ruppel](https://github.com/jeremyruppel)._

Parameters: `none`

Template:

```html
\{{ordinalize 3}}
\{{ordinalize 1}}
\{{ordinalize 22}}
```
Renders to:

```
3rd
1st
22nd
```
